# LinkedIn Scraper

This project provides a Python script for logging into LinkedIn and searching for company information using Selenium. It uses headless Chrome to perform the scraping, ensuring minimal resource usage and invisibility during the operation.

## Features

- Log into LinkedIn using email and password
- Search for a specific company
- Extract and display the company's name and subtitle

## Requirements

- Python 3.6 or higher
- Selenium
- WebDriver Manager
- Beautiful Soup
- Fake UserAgent
